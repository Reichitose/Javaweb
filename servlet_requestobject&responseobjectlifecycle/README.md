关于请求对象和响应对象的生命周期




# 笔记

## 请求对象和响应对象生命周期

1. 在Http服务器接收到浏览器发送的**Http请求协议包**之后，自动为当前的**Http请求协议包**生成一个**请求对象**和一个**响应对象**
2. 在Http服务器调用doGet/doPost方法时，负责将**请求对象**和**响应对象**作为实参传递到方法中，确保方法的执行
3. 在Http服务器准备推送Http响应协议包之前，负责将本次请求所关联的**请求对象**和**响应对象**销毁



> 请求对象和响应对象的生命周期贯穿于一次请求的处理
>
> 请求对象和响应对象相当于用户在服务端的代言人
